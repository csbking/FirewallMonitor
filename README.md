# Firewall Monitor - Cython Version

**Author:** Alex Hunter (CSB Team)

**Description:**  
A real-time firewall monitor script for Termux/Linux, written in Cython.  
Detects suspicious ports, high-risk IP ranges, and unknown public IPs.  

**Usage:**
1. Clone this repository.
2. Run: `python3 setup.py build_ext --inplace`
3. Import and use the firewall monitor.
